5.05 PAYLOAD DESCRIPTOR:

VTXHEN = allows for fake signed content to run, adds debug settings.

MIRAHEN = allows for fake signed content to run, adds debug settings, starts UART for MIRA on port 9999/9998, run it twice for Bin Loader function.

BINLOADER = allows you to send a payload.bin to your PS4.

MIRA+BIN = run once for Mira, twice for Bin Loader.

BLOCKER + FTP = blocks unwanted updates (perma) and starts a temporary ftp server.

BLOCK EX = blocks every possible update location.

LINUX = lets you boot linus kernel from your USB stick to use your PS4 for Linux.

KEXEC = allows you to boot linux again from PS4 OS (orbis)

VTXDUMP = allows you to dump your legit games (disc/psn) to USB hdd, with or without patches, and merged/unmerged.

DB_SG = backs up all critical databases, save games, and trophy data to USB.

APPTOUSB = moves games to any external USB (v3 you can specify mount point), circumventing the internal drive running out of space.

APPTOUSB_ALT = moves games to external USB1, circumventing the internal drive running out of space.

USB_FPKG = alternative based on the pkg2usb by SiSTRO and AppToUsb by stooged,

KERNELDUMP = allows you to dump the kernel to USB (4.05-5.05)

KERNELDUMPER = allows you to dump the kernel to USB.

FS_DUMPER = dumps PS4 Filesystem to USB.

VR_ENABLER = enables VR if you forgot to enable it before running HEN

VTXFTP = starts an ftp server on you PS4 so you can connect and copy/paste files to/from your PC/PS4.

FTP+R/W = same as ftp server but with full R/W privs on all partitions of on you PS4.

REACTPSPLUS = sets the RTC to 2012 so you can play expired games (UNREACTPSPLUS sets back to 2018).

REACTPSPLUSMODDABLE = sets the RTC to any date you want so you can play expired games.

PS4TRAINER = listens for .cht files to be loaded for your games.

PS4CHEATER = listens for .cht files to be loaded for your games.

PS4API = API server for real time modding (RTM)

APII_INTENSE_EU_127 = mod menu for only EU GTAV V1.27

LAMANCE = mod menu for US/EU GTAV V1.00

PSYC0S = mod menu for US/EU GTAV V1.00

APII_INTENSE = mod menu for US/EU GTAV V1.00

WILDEMODZ = mod menu for US/EU GTAV V1.00

TRAINER = mod menu for US/EU GTAV V1.00

NOTANOTHER = mod menu for US/EU GTAV V1.00

ARABICGUY = mod menu for US/EU GTAV V1.00

CACHE_INSTALL= copies ApplicationCache.db from USB root to the browser cache, makes basic cache if USB not found.

USB_CACHE_INSTALL= copies ApplicationCache.db from USB root only.

APP_CACHE = installs an ApplicationCache.db thats a modified version of Al-Azif's PEH to your Browser.

HISTORY_BLOCKER = enable/disable the auto loading of the last page, shows "frequently used pages" instead.
